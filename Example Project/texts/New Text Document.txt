Business process and workflow management platform for businesses of small sizes. The primary purpose of the platform is solving typical tasks related to performance functionality and more. Application is exceptionally simple to use. Start with a blank form. After selecting a form, you can add fields quickly and easily for all of the data you need to collect, using small a pop-up window form editor. The primary purpose of the platform is solving typical tasks related to performance functionality and more. Add or Modify buttons have "pressed" and "released" states. They indicate that some application mode is enabled or disabled. If the action associated with a button is not available, the button appears shaded. The application  supports drop down operations. The ComboBox control provides combined functionality of a text box and a listbox in a single control. Only one list item is displayed at one time in a ComboBox and rest of the available items are loaded in a drop down list.
